API 文档

LLM Proxy API

POST/v1/chat/completions

OpenAI Chat Completions

curl http://127.0.0.1:7300/v1/chat/completions \
  -H "Authorization: Bearer skm-xxx" \
  -H "Content-Type: application/json" \
  -d '{"model":"auto","messages":[{"role":"user","content":"Hello"}]}'
POST/v1/messages

Anthropic Messages

curl http://127.0.0.1:7300/v1/messages \
  -H "x-api-key: skm-xxx" \
  -H "Content-Type: application/json" \
  -d '{"model":"auto","messages":[{"role":"user","content":"Hello"}],"max_tokens":1024}'
GET/v1/models

List models

curl http://127.0.0.1:7300/v1/models -H "Authorization: Bearer skm-xxx"
GET/v1/health

Health check

curl http://127.0.0.1:7300/v1/health
# {"status":"ok","service":"nantianmen"}

Admin API

GET/api/admin/providers

List all providers

POST/api/admin/providers

Create provider (body: name, protocol, base_url, api_key)

PUT/api/admin/providers/{id}

Update provider

DELETE/api/admin/providers/{id}

Delete provider

POST/api/admin/providers/{id}/health

Health check

GET/api/admin/providers/{id}/models

Fetch models

PUT/api/admin/providers/{id}/models/{mid}/default

Set default model

GET/api/admin/api-keys

List all API keys

POST/api/admin/api-keys

Create API key (body: name, note)

DELETE/api/admin/api-keys/{id}

Delete API key

GET/api/admin/stats

Query stats (params: provider_id, model_name, api_key_id, start_time, end_time)
